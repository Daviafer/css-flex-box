---- Flex Container

- display	= inicializador do container
- flex-direction	= fazer direcionamento dos itens (linha, coluna)
- flex-wrap	= quebra de linha 
- flex-flow	= abreviação do flex-direction e flex-wrap

- justify-content = (flex-start; flex-end; center; space-between; space-around) alinhar itens container de acordo com sua direção
- aling-items	= (center; stretch; flex-start; flex-end; baseline) alinhar itens de acordo com o eixo do container
- aling-content	= (flex-start; flex-end; center; stretch; space-between; space-around) alinhar as linhas do container

---- Flex Item

- flex-grow	= define fator de crescimento. Por padrão seu valor é 0, o que indica que o item não deve crescer, e são aceitos apenas valores numéricos positivos.
- flex-basis	= define tamanho inicial do item antes da distribuição
- flex-shrink	= capacidade de redução ou compressão do tamanho de um item
- flex		= abreviação do grow, shrink e basis 

- order		= ordem de distribuição dos itens
- aling-self	= alinhamento específico do container
